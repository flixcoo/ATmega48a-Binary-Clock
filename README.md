### Stunden-LEDs
PD3 - Stunde 
PD4 - Stunde 2<sup>1</sup>

### Minuten-LEDs
PC0 - Minute 2<sup>5</sup>  
PC1 - Minute 2<sup>4</sup>  
PC2 - Minute 2<sup>3</sup>  
PC3 - Minute 2<sup>2</sup>  
PC4 - Minute 2<sup>1</sup>  
PC5 - Minute 2<sup>0</sup>

PB0 - Button 1  
PB1 - Button 2  
PD2 - Button 3
